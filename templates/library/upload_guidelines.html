Upload Guidelines

Please review these guidelines before uploading books to ensure quality and compliance.

Allowed Formats

  • PDF files only — We currently accept PDF format for book uploads.
  • Cover images should be in JPG, PNG, or WebP format.
  • Word documents, EPUB, DJVU, and scanned images are not accepted.

Maximum File Size

50 MB per PDF file

Files exceeding this limit will be rejected. Please compress your PDF if necessary.

Copyright Rules

  • Only upload books that are free to distribute or in the public domain.
  • Classical Islamic texts (Hadith collections, Tafsir, Fiqh manuals) are generally public domain.
  • Do not upload copyrighted books without permission from the publisher or author.
  • Violations may result in content removal and account suspension.

Quality Standards

  • PDF should be text-searchable (not scanned images) when possible.
  • Provide an accurate title, scholar/author name, and category.
  • Include a book description to help users find and understand the content.
  • Adding a cover image greatly improves discoverability.
  • Avoid uploading duplicate books. The system checks for existing copies.
